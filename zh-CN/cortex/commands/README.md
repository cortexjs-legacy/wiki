# Cortex(1)

命令索引。

## 基本命令

### `cortex -h`
### `cortex --help`

查看 cortex 基本命令帮助

### `cortex <command> -h`

查看 `cortex <command>` 的简明帮助。


### `cortex help <command>`

查看 `cortex <command>` 命令的详细帮助，包含使用方法和详细的参数说明。

比如，可以在命令行中运行：

```bash
cortex help install
```
